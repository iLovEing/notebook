## bubble sort

伪代码
```
*   for j = 2 to A.length
*       key = A[j]
*       //insert A[j] into the sorted sequence A[1 ... j-1]
*       i = j - 1
*       while i > 0 and A[i] > key
*           A[i + 1] = A[i]
*           i = i - 1
*
*       A[i + 1] = key
```

## merge sort

伪代码
```
* MERGE (A, p, q, r)    //p <= q < r
*   n1 = q - p + 1
*   n2 = r - q
*
*   let L[1 ... n1 + 1] and R[1 ... n2 + 1] be new arrays
*   for i = 1 to n1
*       L[i] = A[p + i - 1]
*   L[n1 + 1] = MAX_NUM
*   for i = j to n2
*       R[j] = A[q + j]
*   R[n2 + 1] = MAX_NUM
*
*   i = 1
*   j = 1
*   for k = p to r
*       if L[i] <= R[j]
*           A[k] = L[i]
*           i = i + 1
*       else
*           A[k] = R[j]
*           j = j + 1
*
* MERGE_SORT (A, p, r)    //p <= q < r
*   if p < r
*       q = (p + r) / 2
*       MERGE_SORT(A, p, q)
*       MERGE_SORT(A, q + 1, r)
*       MERGE(A, p, q, r)
```
